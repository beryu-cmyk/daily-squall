# Daily Squall Generator

毎日ランダムなシチュエーションでスコールのイラストプロンプトを生成するツール。

## 使い方

1. [Daily Squall Generator](https://あなたのユーザー名.github.io/daily-squall/) にアクセス
2. 「今日のスコールを生成」ボタンをクリック
3. 生成されたプロンプトをコピー
4. お好みの画像生成AI（pixAI、GPTなど）でイラストを生成

## 機能

- 🎲 ランダムな服装・表情・テーマ・絵柄の組み合わせ
- 💍 記念日には特別なシチュエーション
- 📋 ワンクリックでプロンプトをコピー

## 記念日を追加するには

`data.js` の `SPECIAL_DAYS` を編集してください。
